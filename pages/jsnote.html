JS (JavaScript)



Introduction

A lightweight, interpreted, or JIT-compiled programming language with first-class functions.
It is Commonly used for creating interactive effects within web browsers.


Basic Syntax

Comments:

Single-line: //
Multi-line: /* */

Variables: Declared using var, let, or const

Data Types: Number, String, Boolean, Null, Undefined, Symbol (ES6), BigInt (ES2020)

Objects: Collections of properties.

Control Structures

Conditional Statements: if, else if, else, switch

Loops: for, while, do...while, for...in, for...of
