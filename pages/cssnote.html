CSS (Cascading Style Sheets)



Introduction

CSS is a stylesheet language used to describe the presentation of a document written in HTML or XML.
It controls the layout, colors, fonts, and overall appearance of web pages.


Basic Syntax

Selector: Targets the HTML element(s) to be styled.

Property: The aspect of the element to be styled (e.g., color, font-size).

Value: Specifies the setting for the property (e.g., red, 16px).

Types of CSS

Inline CSS : Applied directly to HTML elements using the style attribute.

Internal CSS:Defined within the style tag in the HTML head.

External CSS:Defined in a separate CSS file and linked to the HTML document using the link tag.
